Installation:
- Copy `MWS Handler`, `basic_games` and `installer_non_zipped.py` to your MO2/Plugins/ folder.

Change log:
- 0.0.2:
    - [MWS MO2 Plugin] mws-mo2://install/:game_id/:mod_id/:file_id
    - [Link Handler] Changed protocol from MWS to MWS-MO2.
    - [Link Handler] Uses the elements of the MWS-MO2 link to populate the download metadata file. 

- 0.0.3:
    - [Link Handler] Get mod name (modName=), file version (version=), and mod version (newestVersion=) from the MWS API and put them into the metadata file.
